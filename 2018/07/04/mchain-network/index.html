一、概述

现在北京和深圳两地有办公室，都可以通过VPN接入网络。

北京VPN接入服务器是固定IP61.149.7.240

深圳VPN接入服务器为动态域名sino-cloud.ticp.io

二、创建VPN

1、Mac

打开系统偏好设置，选择网络

点击左下角的“+”号，创建一个新的服务

接口选择VPN，VPN类型选择L2TP/IPSec，服务名称随意输入，点击“创建”按钮

配置选择默认，北京服务器地址输入61.149.7.240，深圳服务器地址输入sino-cloud.ticp.io，帐户名称输入mchainvpn

点击“鉴定设置”按钮，在用户鉴定和机器鉴定处输入密码（另行告知），点击“好”按钮

点击“应用”按钮保存设置

点击“连接”按钮进行测试

如果没有意外，应该显示已连接，北京会分配一个192.168.138网段的IP地址，深圳会分配一个192.168.139网段的IP地址

在终端里面执行命令ping 192.168.138.1，检测是否可以连通网关

三、手工添加路由

通过VPN连接到北京或者深圳后，默认只能访问当地的网络资源，如果需要访问其他地方的网络，还需要手工添加路由信息。

注：北京网络为192.168.138.0/24，深圳网络为192.168.139.0/24，本例连接到深圳VPN，添加路由访问北京网络

1、Mac

连接到深圳的VPN服务器，应该获取到192.168.139网段的IP地址

打开终端，执行ping 192.168.139.1，应该可以连通

打开终端，执行ping 192.168.138.1，应该无法连通

打开终端，执行命令添加路由信息sudo route add 192.168.138.0 192.168.139.1，输入密码，成功

注：连接到北京VPN，应该执行sudo route add 192.168.139.0 192.168.138.1

打开终端，执行ping 192.168.138.1，应该可以连通了

注：可以自行搜索将路由加为永久路由
